# EU English language standards

User-facing English on this site follows **EU English** (British English as used by the European Union institutions). This aligns with the [Interinstitutional Style Guide](https://style-guide.europa.eu/) and the [European Commission English Style Guide](https://commission.europa.eu/resources/translation-and-drafting-resources/guidelines-translation/guidelines-translating-english_en).

## Spelling (EU / British)

- **-ise** not -ize: *organise*, *realise*, *personalise*, *optimise*
- **-isation** not -ization: *organisation*, *optimisation*
- **-our** not -or (where applicable): *colour*, *favour*, *honour*
- **adviser** not advisor: *financial adviser* (and in legal/regulatory contexts)
- **-re** not -er where applicable: *centre* (if used)
- **-ence** not -ense: *licence* (noun), *defence*
- **transport** not transportation (for public transport)
- **towards** not toward

## Punctuation and formatting

- No full stop after **Mr**, **Mrs**, **Ms**, **Dr** in running text: *Mr Williams*
- Single quotation marks for quotations in running text where appropriate; double for direct speech is also acceptable.
- Day–month–year for dates: *14 February 2026* or *14 Feb 2026*

## Reference

## Where this applies

- All strings in `src/lib/i18n/translations.ts` for locale `en`
- Any hardcoded English in `src/app` pages (e.g. careers, work-with-me, legal) that is user-facing

German (`de`) is unchanged; this note applies only to the English locale.
